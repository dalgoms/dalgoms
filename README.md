## Growth Marketing OS

분석부터 크리에이티브, 실행, 측정까지 — 직접 구축한 도구들이 하나의 마케팅 운영 시스템으로 연결됩니다.

```mermaid
flowchart LR
  A["Analyze\nWebScout"] --> B["Create\nAd Creative Tool"]
  B --> C["Automate\nai_process"]
```

| Phase | 도구 | 역할 | 상태 |
|---|---|---|---|
